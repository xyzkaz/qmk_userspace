# Ploopy Nano 2 Trackball Configuration

My custom configuration for Ploopy Nano 2 Trackball

## Behaviors

- Click -> left mouse click
- Hold -> activate scroll mode

### For working with keyboard

- Scroll lock on -> activate scroll mode
- Num lock on -> use first DPI
- Num lock off -> use second DPI

I setup my keyboard to turn on Scroll lock when holding down F13, turn on Num lock when holding down Right Ctrl.
